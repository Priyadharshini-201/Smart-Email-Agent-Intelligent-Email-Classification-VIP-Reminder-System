Overview

Smart Email Agent is an intelligent, rule-based system that automatically classifies emails into categories and generates reminders for high-priority messages. It uses a multi-agent architecture to parse emails, classify them, generate VIP reminders, and produce reports with visualizations.

This project is designed for professionals or students who want to efficiently manage emails without relying on cloud AI services.

Features

Email Classification:

Categories: VIP, Meeting, Project, Follow-up, General

VIP detection based on sender (e.g., CEO, boss) or keywords (urgent/important)

VIP Reminders:
Automatically generates reminders for VIP emails.

Reporting & Visualization:

Text report of email counts per category

Bar chart visualization of email categories

Multi-Agent System:

Parser Agent: Loads and processes email data

Reminder Agent: Generates VIP email reminders

Report Generator Agent: Creates reports and charts
